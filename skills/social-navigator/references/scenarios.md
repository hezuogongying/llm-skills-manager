# 常见社交场景模板

## 场景1：寻求职业机会

### 配置示例
```
目标：获得内推机会或职位信息
对方：某公司员工/HR/业内前辈
关系：弱关系（朋友介绍/LinkedIn连接）
```

### 路径演进

**阶段1：建立连接**（距离远，信息模糊）
- 操作：自我暴露 + 开放提问
- 目的：让对方愿意聊，了解对方现状
- 示例："我是通过XX认识您的，看到您在XX领域深耕多年，很想请教一下目前行业的情况"

**阶段2：信息收集**（距离中，逐渐清晰）
- 操作：开放提问 + 共情确认
- 目的：了解对方公司情况、是否有机会
- 示例："您觉得现在团队最缺什么样的人？"

**阶段3：价值交换**（距离近，路径清晰）
- 操作：价值提供 + 轻推试探
- 目的：展示自己能提供的价值，试探意愿
- 示例："我之前做过XX项目，这方面经验可能对团队有帮助，不知道方便帮忙引荐一下吗？"

### 常见风险信号
- "最近公司在收缩" → 降低期望，转向信息收集
- "我帮你问问" → 积极信号，跟进但不催
- 已读不回 → 等待3天后轻推一次

## 场景2：商业合作开拓

### 配置示例
```
目标：建立合作关系/达成交易
对方：潜在客户/合作方决策者
关系：陌生人（Cold outreach）
```

### 路径演进

**阶段1：突破防备**（距离远，对方有防御）
- 操作：价值前置 + 开放提问
- 目的：给对方不删除/不拒绝的理由
- 示例："我注意到贵司在XX方面的动作，我们刚帮XX公司解决了类似问题，效果是XX。不知道您这边有没有类似的痛点？"

**阶段2：需求确认**（距离中，防备降低）
- 操作：封闭提问 + 复述确认
- 目的：明确对方具体需求和决策链
- 示例："所以您主要是想解决XX问题，对吗？这个事情是您直接决定还是需要其他人参与？"

**阶段3：方案推进**（距离近，双方意愿明确）
- 操作：锚定退让 + 价值提供
- 目的：推动具体方案
- 示例："基于您的需求，我们有两个方案...考虑到您的预算，方案B可能更合适"

### 常见风险信号
- "发个资料我看看" → 可能是敷衍，发资料后约具体时间跟进
- "我们有供应商了" → 探测痛点，是否有未被满足的需求
- "预算有限" → 可能是真的，也可能是谈判策略，需要探测

## 场景3：获取专业建议

### 配置示例
```
目标：获得具体问题的指导建议
对方：行业专家/前辈/导师
关系：有过接触/朋友介绍
```

### 路径演进

**阶段1：尊重式开场**
- 操作：自我暴露 + 具体问题
- 目的：展示已做功课，尊重对方时间
- 示例："我在XX问题上卡住了，自己尝试了ABC方法，但效果不好。想请教您的看法"

**阶段2：深入探讨**
- 操作：开放提问 + 复述确认
- 目的：理解建议背后的逻辑
- 示例："您提到XX很重要，能展开说说是什么让您这么认为的吗？"

**阶段3：闭环感谢**
- 操作：价值提供（反馈）
- 目的：建立长期关系
- 示例："按您说的方法试了一下，效果确实好很多，特别是XX那个点..."

### 关键原则
- 问题要具体，不要"请教一下职业发展"
- 展示你已经思考过，不是伸手党
- 事后反馈是最好的回报

## 场景4：化解冲突/挽回关系

### 配置示例
```
目标：修复受损的关系
对方：客户/同事/合作伙伴
关系：原本良好，因某事受损
```

### 路径演进

**阶段1：承认问题**（对方可能有情绪）
- 操作：共情确认 + 自我暴露
- 目的：让对方感受到被理解
- 示例："这件事确实是我们没做好，给您带来了很大的麻烦，我非常理解您的失望"

**阶段2：理解诉求**
- 操作：开放提问
- 目的：明确对方真正在意什么
- 示例："除了XX问题，还有什么是您希望我们改进的？"

**阶段3：补救方案**
- 操作：价值提供 + 封闭提问
- 目的：给出具体解决方案并确认
- 示例："我们计划这样补救...这样可以接受吗？"

### 关键原则
- 不要急于解释/辩护
- 先情绪，后逻辑
- 承诺要具体可执行

## 对话DOM解析

**情感状态**: 中性偏积极
- 主动提供招聘信息（愿意帮忙）
- 但在筛选/试探（"你做过吗""能接受吗"）

**显性需求**: 
- 需要有大模型经验的人
- 担心候选人无法适应高强度工作

**隐性线索**:
- "我们组压力挺大" → 可能团队人手紧张
- 主动问而非等你问 → 有一定推荐意愿

**可操作节点**:
- 节点1: "有大模型经验的" → 封闭提问确认具体要求
- 节点2: "经常加班" → 共情确认 + 展示态度
- 节点3: 隐含的人手紧张 → 价值提供（我能快速上手）

**风险信号**: 无明显风险，但需要展示匹配度

## 路径评估

**距离目标**: 近（对方已主动透露机会）
**路径收敛度**: 清晰（需求明确：大模型经验+能加班）
**当前阶段**: 价值交换

## 推荐策略

**核心操作**: 价值提供 + 封闭提问
**操作目的**: 证明自己匹配需求，推动下一步
